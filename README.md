# Projet EDF
============
Le but du projet est de produire un programme permettant de simuler :

- Modele Factoriel Schwartz Gaussien
- Modele Hyperbolique Generalise
- Prime de Risque et Options

Projet EDF UMPC 2004, Université Paris VI France, [Theses, O. Senhadji El Rhazi](https://tel.archives-ouvertes.fr/tel-01086146).
